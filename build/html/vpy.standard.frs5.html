pressure_cal(res)[source]¶

Calculates the FRS5 calibration pressure from lb indication. The equation is:

\[p=\frac{r}{r_{cal}} m_{cal}\frac{g}{A_{eff}}\ \frac{1}{corr_{rho}corr_{tem}} + p_{res}\]

with

\[corr_{rho} = 1 - \frac{\rho_{gas}}{\rho_{piston}}\]

and

\[corr_{tem} = 1 + \alpha \beta (\vartheta - 20)\]

vpy.standard.frs5.std module¶

class vpy.standard.frs5.std.Frs5(doc)[source]¶

Bases: vpy.standard.standard.Standard

Calculation methods of large area piston gauge FRS5.

  • p_frs … abs. pressure of the piston gauge [p]=Pa
  • r … reading [r] = lb
  • u_b … standard uncertainty of the balance [u_b] = lb
  • u_sys … sys. uncertanty of the balance [u_sys]=lb
  • r_0 … zero reading [r_0] = lb
  • A … effective area [A]= m^2
  • p_res … residual pressure [p.res]=Pa
  • m_cal … calibration mass piece [m.cal]=kg
  • rho_gas … density gas
  • rho_frs … density frs piston
  • g … accelaration [g]=m/s^2
  • r_cal … indication at m_cal [r_cal]=g
  • r_cal_0 … indication at zero [r_cal_0]=g
  • ab … temperatur coeff. [k]=1/K
  • t … temperature of balance [t]=C

model equation:

\[p=\frac{r_{ind}-r_0+u_b+u_{sys}}{r_{cal}-r_{cal 0}}\ m_{cal}\frac{g}{A_{eff}}\frac{1}{corr_{rho}corr_{tem}}\]

define_model()[source]¶

Defines symbols and model for FRS5. The order of symbols must match the order in gen_val_arr:

  1. A
  2. r
  3. r_zc
  4. r_zc0
  5. r_cal
  6. r_cal0
  7. ub
  8. usys
  9. p_res
  10. m_cal
  11. g
  12. T
  13. rho_frs
  14. rho_gas
  15. ab

The equation is:

\[p=\frac{r}{r_{cal}} m_{cal}\frac{g}{A_{eff}}\ \frac{1}{corr_{rho}corr_{tem}} + p_{res}\]

with

\[corr_{rho} = 1 - \frac{\rho_{gas}}{\rho_{piston}}\]

and

\[corr_{tem} = 1 + \alpha \beta (\vartheta - 20)\]

uncert_rho_frs(res)[source]¶

Calculates the uncertainty of the buoyancy correction for the piston.

Note

In order to avoid an uncertainty value of 0 the value 0 for rho_gas (used during calculation) is replaced by the value at p, N2 and 27°C:

(28.013e-3/(296.0+5.)/8.314*100./2.)
