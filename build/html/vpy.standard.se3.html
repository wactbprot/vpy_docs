class vpy.standard.se3.std.Se3(doc)[source]¶

Bases: vpy.standard.standard.Standard

Configuration and methodes of static expansion system SE3.

There is a need to define the no_of_meas_points: the time: amt_fill (absolut measure time of filling pressure) is used for this purpose if doc is a calibration. For the analysis of state measurements amt is used.

define_model()[source]¶

Defines symbols and model for the static expansion system SE3. The order of symbols must match the order in gen_val_arr:

# . f # . p_fill # . V_5 # . V_start # . p_before # . p_after

The equation is:

\[p = f_{corr} p_{fill}\]

with

\[f_{corr} = \frac{1}{ \frac{1}{f} + \frac{V_{add}}{V_{start}}}\]
